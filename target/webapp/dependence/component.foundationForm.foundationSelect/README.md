## 注意：
##### 如果使用的AUI2是4.4和5.0版本，下载更新组件后需要执行以下操作：
- 把js文件夹里面的js文件替换更新到AUI2/dependence/AUI/js/里面
- 把css文件夹里面的css文件替换更新到AUI2/dependence/AUI/css里面


## 绕坑点
- 校验时，元素必须选择 `下拉框容器（推荐）`、类型选择`值（改变）`;
